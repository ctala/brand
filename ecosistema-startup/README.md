## 🎯 Propósito de Marca

El medio #1 en español para el ecosistema startup latinoamericano. Noticias, convocatorias y contexto para que los founders tomen mejores decisiones.

**Posicionamiento:** Medio de referencia, no marca personal.  
**Tagline:** "El Ecosistema Startup — Noticias y recursos para founders LATAM"

---

## 🗣️ Voz y Tono

### Voz (constante)
- **Periodístico, no panfletario:** Narra los hechos con contexto, no exagera ni sensacionaliza.
- **Founder-first:** Siempre desde la perspectiva de qué significa esto para un founder o inversor.
- **Conciso:** No hay palabras de relleno. Cada frase agrega valor.
- **Neutro en 1ª persona:** Nunca "yo" o "Cristian". Siempre "el ecosistema", "los founders", "la startup".

### Tono por tipo de contenido
| Tipo | Tono |
|------|------|
| Noticias de rondas | Informativo, con contexto "¿qué significa esto?" |
| Convocatorias | Práctico, claro, con fechas y CTA a aplicar |
| Tendencias | Analítico, datos, perspectiva LATAM |
| Glosario | Educativo, simple, sin jerga innecesaria |
| LinkedIn ECO | Más conversacional, pregunta al final para engagement |

### Lo que NUNCA hace la voz de Ecosistema
- ❌ Referencias a Cristian Tala en primera persona
- ❌ Menciones a Pago Fácil como "mi empresa"
- ❌ "Yo vendí", "yo invertí", "mi experiencia" → Es un medio, no una persona
- ❌ Opiniones personales → Solo hechos + contexto para founders
- ❌ Datos sin fuente → Todo dato factual debe tener fuente verificada
- ❌ Atribuir contenido a "viral en inglés" o fuentes externas en notas → Adaptar con ángulo propio LATAM y atribuir al equipo editorial
- ❌ Datos de rondas o fondos inventados/estimados → SIEMPRE fuente verificada o no publicar

## ✅ Checklist de Calidad de Contenido (Ecosistema)

### Contenido
- [ ] **Sin referencias personales de Cristian:** ¿Hay "yo", "Pago Fácil", "mi empresa"? → Eliminar
- [ ] **Fuente verificada:** ¿Todo dato de ronda/noticia tiene fuente pública citada?
- [ ] **No inventado:** ¿Montos, inversores y fechas son verificados? → NO usar placeholders
- [ ] **Contexto LATAM:** ¿La noticia tiene ángulo para founders latinoamericanos?
- [ ] **Voz de medio:** ¿Suena a medio editorial o a persona? → Debe sonar a medio

### NocoDB — Columnas correctas
- [ ] **Content:** Texto limpio publicable. Sin markdown (##, **). Sin notas internas. Texto final tal como aparece en la red social o WordPress.
- [ ] **Content Preview:** Resumen 1 línea del tema/ángulo (para revisión rápida)
- [ ] **Notes / Reviewer Notes:** Notas internas, contexto editorial, fuentes. NUNCA en Content.
- [ ] **Asset URL:** URL de imagen en R2 (verificar que existe y carga)
- [ ] **Has Media:** `true` si hay imagen

### Media
- [ ] **Paleta ECO:** Naranja + verde ECO, NO Synthwave
- [ ] **Imagen existe:** URL carga correctamente
- [ ] **Sin datos inventados en la imagen:** Texto en imagen debe coincidir con copy
- [ ] **Logo ECO visible:** Debe tener branding de Ecosistema
