# Assets — Es la Hora de Aprender

Los assets del podcast viven en el **repo del sitio** (`ctala/landing-es-la-hora-de-aprender`), no en R2. El manual de marca se sirve desde `https://assets.cristiantala.com/brand/hora.html`.

## Thumbnails de Episodios

Ubicación: `public/thumbnails/` en el repo del sitio. 4 variantes por episodio.  
Formato principal: WebP, 1280×720px. No hay listado de directorio: se accede por URL directa.

| Variante | Patrón de URL |
|----------|---------------|
| Full WebP | `https://eslahoradeaprender.com/thumbnails/ep{NN}.webp` |
| Small | `…/thumbnails/ep{NN}-sm.webp` |
| Extra-small | `…/thumbnails/ep{NN}-xs.webp` |
| JPG fallback | `…/thumbnails/ep{NN}.jpg` |
